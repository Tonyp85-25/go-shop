# Go Shop

Go shop is an ecommerce web app (for learning purpose) written in Go 

## Tools used
- Go 1.26
- Gin Framework
- GORM
- Zerolog
- Postgresql


## Development tools
- [Golangci](https://golangci-lint.run) : for managing linters and formatters
- [LocalStack](https://docs.localstack.cloud/aws/) : for emulating AWS services locally
- [Devbox](https://www.jetify.com/devbox) : for managing dev environements without containers
- [Taskfile](https://taskfile.dev): for managing development tasks
